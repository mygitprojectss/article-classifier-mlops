# Article Classification using ML, Deep Learning and Transformer

This project classifies news articles using three different approaches:

• Machine Learning  
• Deep Learning  
• Transformer Models  

Users can enter an article and get predictions from the trained models.

## Features

- Text preprocessing
- ML model prediction
- Deep learning model prediction
- Transformer based classification
- Streamlit interactive UI

## Tech Stack

- Python
- Scikit-learn
- TensorFlow / PyTorch
- HuggingFace Transformers
- Streamlit
